## 一、每日操作流程（5分钟版）

### 早上：查看进度
```
打开这个文件 → 看到当前状态
```

### 白天：等Agent干活
```
不需要做任何事
```

### 晚上：检测构建
```
1. 下载最新构建包
2. 运行exe 1-2分钟
3. 确认核心功能正常
4. 如有问题，记录反馈
```

## 十、遇到问题？

### Q: Agent卡住了怎么办？
A: 查看 `code_automation.py status`，如果一直卡住，重启：
```powershell
python code_automation.py start --project 游戏名 --game 游戏名
```

### Q: 需要修改任务怎么办？
A: 手动编辑 `D:\GJ\docs\dispatch\task_queue.json`

### Q: 美术资源没来？
A: 在群里提醒美术队友

### Q: 想看代码？
A: **不要看**。如果你看了，就要自己做决策，违背了零代码接触的原则。
