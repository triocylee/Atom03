# Vampire Survivors 自动化流水线

## 流水线概述

```
用户输入: 游戏名
     ↓
┌─────────────────────────────────────────────────────────────┐
│  Phase 1: 游戏分析与任务分解（自动化）                         │
│  - librarian: 分析游戏核心玩法                               │
│  - 生成三角色任务包                                           │
└─────────────────────────────────────────────────────────────┘
     ↓
┌─────────────────────────────────────────────────────────────┐
│  Phase 2: 任务派发（人工）                                   │
│  - 美术任务 → 派发给美术队友                                  │
│  - 测试任务 → 派发给测试队友                                  │
│  - 代码任务 → 进入自动化制作                                   │
└─────────────────────────────────────────────────────────────┘
     ↓
┌─────────────────────────────────────────────────────────────┐
│  Phase 3: 代码自动制作（自动化）                              │
│  - deep agent集群并行实现各模块                               │
│  - 代码生成 → 本地Unity项目                                  │
└─────────────────────────────────────────────────────────────┘
     ↓
┌─────────────────────────────────────────────────────────────┐
│  Phase 4: 人工检测（自动化打包后）                            │
│  - 用户拉取最新构建                                           │
│  - 人工测试核心功能                                           │
│  - 反馈问题 → 进入下一轮修复                                   │
└─────────────────────────────────────────────────────────────┘
```
