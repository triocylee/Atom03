# Raft - M1 美术任务清单

## 本阶段目标
- 先交可运行资源，让代码能挂进去，让测试能看到画面。
- 这一阶段不追求精细，只要求统一、可用、能导入。

## 本阶段需要准备的资源

### 模型类
- 玩家第一人称手部：1套
- 钩子：1个
- 木筏基础地板：1个
- 漂浮资源：4个
- 桶：1个

### 贴图和UI类
- 生命图标：1个
- 饥饿图标：1个
- 口渴图标：1个
- HUD底板或状态条底图：1套

### 材质类
- 海面基础材质：1套
- 木筏基础材质：1套
- 漂浮物通用材质：1套

## 交付说明
- 每次交资源时写清楚资源名、用途、版本。
- 如果某个资源还不是最终版，标注“占位版”。
- 如果有缩放或朝向要求，必须单独备注给代码。

## 需要等待的内容
- 本阶段基本不用等代码才能开始。
- 如果代码补充了木筏地板尺寸、挂点朝向，要按代码规格修正一次。

- 资源能正常导入项目。
