# Raft - M1 测试任务清单

## 本阶段目标
- 建立M1冒烟检查标准。
- 确保代码和美术每交一轮内容后，都能快速确认主流程有没有被破坏。

## 本阶段测试重点
- 游戏是否能启动
- 是否能进入主场景
- 玩家能否移动和跳跃
- 玩家能否站在木筏上
- 海面、木筏、漂浮资源是否正常显示
- HUD是否正常显示
- 日志是否能帮助定位问题

## M1冒烟清单

### 启动类
- 游戏是否能正常启动
- 是否能进入主场景
- 是否出现黑屏
- 是否出现启动后闪退

### 玩家控制类
- 鼠标能否控制视角
- 键盘移动是否正常
- 跳跃是否正常
- 玩家是否会卡死或无限下坠

### 场景显示类
- 海面是否可见
- 木筏是否可见
- 漂浮资源是否可见
- 天空或环境是否异常丢失

### HUD类
- 生命区域是否显示
- 饥饿区域是否显示
- 口渴区域是否显示
- HUD是否遮挡视角过多

### 资源显示类
- 玩家手部是否显示正常
- 钩子是否显示正常
- 漂浮资源是否存在明显比例错误
- 木筏地板是否明显尺寸错误

### 日志类
- 启动是否有日志
- 进入场景是否有日志
- 漂浮资源生成是否有日志
- 交互检测是否有日志
